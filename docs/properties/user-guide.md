# Guia do Usuário: Gestão de Imóveis e Mídia (imobWeb)

Bem-vindo ao sistema de gestão de imóveis mais avançado do mercado. Este guia explica como utilizar as funções de cadastro, otimização de imagem e inteligência artificial.

## 1. Cadastrando um Imóvel
O imobWeb utiliza um sistema inteligente que adapta o formulário ao tipo de imóvel:
1. Clique em **Novo Imóvel**.
2. Selecione o **Tipo de Imóvel** (ex: Apartamento, Galpão, Fazenda).
3. O sistema habilitará automaticamente campos específicos (ex: para Apartamentos, aparecerá "Andar" e "Valor do Condomínio").
4. Preencha as informações básicas e localização.

## 2. Gestão de Imagens e Galeria
As fotos são o ativo mais importante do seu anúncio.
- **Upload em Massa**: Você pode arrastar e soltar dezenas de fotos simultaneamente.
- **Otimização Automática**: O sistema converte suas fotos para formatos leves (WebP/AVIF) mantendo a nitidez total.
- **Ordenação Inteligente**: Clique em "Sugerir Ordem via IA" para que o sistema coloque as fotos mais impactantes primeiro.

## 3. Usando a Inteligência Artificial (AIBoost)
Disponibilizamos ferramentas de IA para aumentar sua conversão:
- **Melhoria de Foto**: Na galeria, clique em "Melhorar com IA". O sistema ajustará cores, brilho e nitidez automaticamente.
- **Relatório de Qualidade**: O sistema dá uma nota de 0 a 100 para cada foto. Fotos com nota baixa (abaixo de 60) devem ser substituídas para não prejudicar o anúncio.
- **Gerador de Descrição**: Com base nas fotos enviadas, clique em "Gerar Descrição IA". O sistema escreverá um texto persuasivo destacando os pontos fortes do imóvel.

## 4. Visualização e Apresentação
Você pode alternar como seus clientes veem os imóveis:
- **Grid de Luxo**: Ideal para navegação rápida.
- **Mapa Interativo**: Para busca por localização precisa.
- **Tour Virtual**: Integre links de Matterport ou fotos 360º facilmente na aba "Mídia".

## 5. Dicas de Performance
Para que seu site carregue instantaneamente e ranqueie bem no Google:
- Use sempre imagens de alta resolução (o sistema cuidará da compressão).
- Preencha as tags de IA sugeridas para facilitar a busca dos clientes.
- Mantenha pelo menos 5 fotos de alta qualidade por imóvel.
