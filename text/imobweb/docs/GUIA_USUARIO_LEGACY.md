## Introdução

O **imobWeb** é uma plataforma completa de CRM imobiliário que ajuda corretores e proprietários a gerenciarem seus imóveis, leads e atividades de forma eficiente.

### Diferenças de Uso

| Recurso | Corretor | Proprietário |
|---------|----------|--------------|
| Gerenciar múltiplos imóveis | ✅ | ✅ (próprios) |
| Ver todos os leads | ✅ | ❌ |
| Chat com IA | ✅ | ✅ |
| Dashboard completo | ✅ | Limitado |
| Sugestão de preço | ✅ | ✅ |
| Geração de descrição | ✅ | ✅ |
| Agendar visitas | ✅ | ✅ |

## Para Corretores

### 1. Cadastro e Configuração

#### Primeiro Acesso
1. Acesse o sistema
2. Faça login ou crie uma conta
3. Complete seu perfil (nome, telefone, CRECI se aplicável)
4. Configure suas preferências

#### Dashboard
O dashboard mostra:
- **Imóveis Ativos**: Total de imóveis publicados
- **Leads Novos**: Contatos recentes
- **Visitas Agendadas**: Próximas visitas
- **Performance**: Métricas do mês

### 2. Gerenciamento de Imóveis

#### Adicionar Novo Imóvel
1. Clique em "Novo Imóvel"
2. Preencha as informações:
   - **Tipo**: Apartamento, casa, terreno, etc.
   - **Endereço**: Complete com CEP, rua, número, bairro, cidade
   - **Características**: Quartos, banheiros, vagas, área
   - **Preço**: Valor de venda ou aluguel
   - **Descrição**: Use a IA para gerar!
3. Adicione fotos de qualidade
4. Publique ou salve como rascunho

#### Usar a IA para Preço
1. No formulário do imóvel, clique em "Sugerir Preço"
2. O sistema analisará:
   - Localização
   - Tamanho
   - Características
   - Dados de mercado
3. Accept ou ajuste o valor sugerido

#### Gerar Descrição Automática
1. Clique em "Gerar Descrição"
2. Selecione o tom (formal, informal, luxo)
3. Adicione highlights
4. Escolha a melhor versão
5. Edite se necessário

### 3. Gestão de Leads

#### Tipos de Lead
- **Novo**: Contato recente
- **Qualificado**: Verificado interesse
- **Em negociação**: Negociação ativa
- **Fechado**: Negócio concretizado
- **Perdido**: Sem interesse

#### Ações com Leads
- **Contatar**: Registrar contato
- **Qualificar**: Avaliar potencial
- **Agendar**: Marcar visita
- **Proposta**: Enviar proposta
- **Finalizar**: Fechar ou perder

#### Usar o Chat de IA
1. Acesse "Chat IA" no menu
2. Converse naturalmente:
   - "Tenho um lead novo"
   - "Preciso qualify esse cliente"
   - "Agendar visita para amanhã"
3. A IA ajuda com sugestões

### 4. Agendamento de Visitas

#### Criar Agendamento
1. Selecione o lead ou cliente
2. Escolha o imóvel
3. Defina data e horário
4. Adicione observações
5. Confirme o agendamento

#### Lembretes
- Notificação 1 dia antes
- Notificação no dia
- Confirmação automática

### 5. Relatórios

#### Relatório Semanal
Receba toda semana:
- Novos leads
- Visitas realizadas
- Propostas enviadas
- Fechamentos do período

#### Exportar Dados
Exporte paraplanilha:
- Lista de imóveis
- Lista de leads
- Histórico de atividades

## Recursos de IA

### Sugestão de Preço

**Como usar:**
1. Acesse o imóvel
2. Clique em "IA" → "Sugerir Preço"
3. Preencha características
4. Receba a sugestão

**Entenda os resultados:**
- **Preço Sugerido**: Melhor preço estimado
- **Faixa**: Mínimo e máximo aceitável
- **Confiança**: Quão preciso é o cálculo
- **Fatores**: O que influencia o preço

**Dicas:**
- Atualize dados para preços mais precisos
- Considere a faixa, não só o valor central
- Compare com imóveis similares

### Geração de Descrição

**Modelos disponíveis:**

| Modelo | Uso |
|--------|-----|
| Curto | Cards, anúncios rápidos |
| Médio | Listagens, portais |
| Completo | Anúncios detalhados |
| Variações | Testes A/B |

** Tons:**

- **Formal**: Imóveis de luxo, empresas
- **Informal**: Mídias sociais, younger público
- **Persuasivo**: Vendas rápidas
- **Técnico**: Investidores
- **Luxo**: Imóveis premium

### Chat Inteligente

**O que pode fazer:**

- ✅ Sugerir preço de imóvel
- ✅ Gerar descrições
- ✅ Qualificar leads
- ✅ Agendar visitas
- ✅ Responder dúvidas
- ✅ Dar sugestões de marketing

**Como conversar:**

```
Usuário: Preciso ajuda com um imóvel
IA: Claro! O que você precisa?
- Sugerir preço
- Criar descrição
- Agendar visita
- Qualificar lead
```

## Dicas e Truques

### Para Corretores

1. **Use a IA diariamente**
   - Gere descrições melhores
   - Precifique corretamente
   - Qualifique leads rápido

2. **Mantenha dados atualizados**
   - Preços de mercado mudam
   - Atualize características
   - Adicione novas fotos

3. **Responda leads rapidamente**
   - Use modelo de resposta
   - Configure respostas rápidas
   - Use o chat de IA

4. **Acompanhe métricas**
   - Verifique semanalmente
   - Identifique padrões
   - Ajuste estratégias

### Para Proprietários

1. **Mantenha contato**
   - Responda rapidamente
   - Informe mudanças
   - Compartilhe feedbacks

2. **Valorize o imóvel**
   - Mantenha limpo
   - Tire fotos profissionais
   - Documente características

3. **Use a IA**
   - Pergunte sobre preço
   - Descubra valor de mercado
   - Compare com vizinhos

|-------|------|

| Ctrl+L | Novo lead |
| Ctrl+E | Exportar |
| Ctrl+F | Buscar |

## Solução de Problemas

### Problema: Não recebo notificações

**Soluções:**
1. Verifique configurações de notificação
2. Permita notificações no navegador
3. Verifique conexão com internet
4. Atualize o app

### Problema: IA não responde

**Soluções:**
1. Verifique sua conexão
2. Tente novamente em alguns segundos
3. Simplique a pergunta
4. Limpe cache do navegador

### Problema: PWA não instala

**Soluções:**
1. Use Chrome ou Safari
2. Atualize o navegador
3. Acesse via HTTPS
4. Limpe dados de navegação
