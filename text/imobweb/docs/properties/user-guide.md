# Guia do Usuário - Módulo de Imóveis & Mídia (imobWeb 2026)

Bem-vindo ao sistema de gestão imobiliária mais avançado do mundo. Este guia ajudará você a extrair o máximo das funcionalidades de Inteligência Artificial e Otimização do imobWeb.

## 1. Cadastrando seu Primeiro Imóvel
Ao iniciar o cadastro, nossa IA detecta automaticamente o tipo de imóvel baseado na sua descrição parcial ou primeiras fotos.
- **Tipos de Imóveis**: O sistema suporta mais de 50 categorias (Residencial, Comercial, Rural, etc.).
- **Filtros Inteligentes**: Os campos de formulário se adaptam ao tipo (ex: Fazendas mostram 'Área de Pastagem', Apartamentos mostram 'Andar').

## 2. Gestão de Imagens de Alta Performance
O imobWeb utiliza um motor de compressão **AVIF/WebP** no servidor.
- **Upload em Massa**: Arraste 50+ fotos de uma vez. O sistema fará a compressão automática sem perda de qualidade perceptível.
- **Ordenação Automática (Smart Sort)**: Nossa IA analisa o conteúdo das fotos e gera a ordem ideal (Capa impactante -> Ambientes principais -> Detalhes).
- **Melhoria com IA (Enhancement)**: Fotos enviadas com baixa iluminação podem ser "corrigidas" automaticamente pelo sistema.

## 3. Relatório de Qualidade Visual
Após o upload, verifique o **Índice Visual** do seu imóvel.
- **Pontuação 0-100**: Mede o potencial de conversão do anúncio.
- **Sugestões**: O sistema dirá se faltam fotos de ambientes específicos ou se a resolução está abaixo do padrão 4K.

## 4. Gerador de Descrição com IA
Não perca tempo escrevendo textos longos.
- Clique em **"Gerar com IA"**.
- O sistema lerá os dados técnicos e *interpretará* as fotos (ex: "Cozinha com bancada em ilha") para criar um texto sedutor e otimizado para SEO.

## 5. Visualizações Dinâmicas
No portal, seus clientes podem alternar entre:
- **Grade Clássica**: Para uma varredura rápida.
- **Mapa Interativo**: Para busca por geolocalização exata.
- **Tour 360**: Imersão total nos ambientes.
