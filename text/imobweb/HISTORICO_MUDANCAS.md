# 📜 Changelog: A Evolução da imobWeb

Toda grande jornada começa com um `git init`. Aqui está o rastro da revolução que estamos construindo.

## [1.2.0] - 2026-04-10 (Lançamento da imobEngine 2.0)
### ✨ Novidades
- **20 IAs Ativadas**: Finalizamos a integração de todas as 20 ferramentas de inteligência artificial prometidas.
- **Staging Virtual Pro**: Agora com suporte a estilos de luxo e industrial.
- **Voice-to-Listing**: Adição do suporte ao dialeto regional do Brasil (Norte/Nordeste/Sul).
- **Match Engine**: Nova versão 40% mais precisa nas recomendações.

### 🛠️ Melhorias
- **Edge Deployment**: Reduzimos o tempo de resposta em 150ms ao mover a lógica de tradução para a borda.
- **Next.js 16**: Migração completa para a versão canary mais estável com suporte a React 19 final.

## [1.1.0] - 2026-03-15 (White-Label Power)
### ✨ Novidades
- **Extreme White-Label**: Lançamento do motor de domínios customizados e proxy de SSL.
- **Multi-Tenant RLS**: Implementação de Row Level Security no PostgreSQL para isolamento total de dados.
- **WhatsApp Cloud API**: Integração nativa com a API oficial da Meta.

## [1.0.0] - 2026-02-01 (O Nascimento)
### ✨ Novidades
- Versão inicial do CRM com cadastro de imóveis, gestão de leads e dashboard básico.
- Integração básica com portais via XML VRSync.
